# 📚 Bases Legales y Técnicas — Sistema IA para Adultos Mayores en Chile

> Repositorio de documentación para el desarrollo de un sistema web con IA (basado en Claude/Anthropic) orientado a apoyar a adultos mayores de Chile en procesos de reconstrucción de hogares, acceso a beneficios sociales y orientación sobre trámites del Estado.
